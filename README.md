# 🎮 Videojuegos-2DGame

Proyecto de desarrollo de videojuegos 2D (réplica de Castlevania) creado con el objetivo de practicar mecánicas, diseño de niveles y programación orientada a videojuegos.

---

## 📌 Descripción

Este repositorio contiene un videojuego 2D desarrollado como proyecto de aprendizaje y experimentación en desarrollo de videojuegos.

El proyecto incluye:

- Movimiento de personajes
- Sistema de colisiones
- Escenarios 2D
- Sprites y animaciones
- Mecánicas básicas de juego
- Gestión de entradas del usuario
- Lógica de enemigos y/o obstáculos

---

## 🕹️ Características

✅ Juego en 2D  
✅ Sistema de físicas y colisiones  
✅ Controles por teclado  
✅ Diseño modular del código  
✅ Sprites y animaciones  
✅ Escenarios interactivos
